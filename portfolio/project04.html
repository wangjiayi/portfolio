Word Game

Technology: JAVA and Java AWT

This is group project！This project is to implement a board. Using Java package Swing and Awc. In this project, the user can play letter games. So when a letter appears on the screen, user need to press the key on the keyboard and if the user input right one he or she can get score，if wrong one，will minus score. After done the game user can save their score into file and load the sorted score list from a file.

The steps for the project:

  • Create User interface, and Iframe. In the frame I have some panels. There are three panels in this game, the first one is user information, the second one is menu pannel and the third one is score pannel which will always pop the highest ten score.
  • Add event listener for buttons , fields and keyboards. And then upload the score.
  • Need to sort the score for the users, and the panel will appear the top 10 scores for the user. Then export it to an executable jar file.
