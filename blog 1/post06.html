Responsive Layout

May 25th, 2017

This website use Flex Box. Lots of layout easily solved with flexbox, so it’s a great example. It use “display” defines a flex container. inline or block depending on the given value. It enables a flex context for all its direct children. And use “flex-direction” establishes the main-axis, thus defining the direction flex items are placed in the flex container. Also use “flex-grow” defines the ability for a flex item to grow if necessary. Moreover, it use “flex-basis” defines the default size of an element before the remaining space is distributed. But I think Flexbox is fine, but It is still not valid for a simple perfect “product grid” with no margins at first and last elements in row.
